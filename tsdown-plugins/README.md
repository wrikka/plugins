# TSDown Plugins

Collection of TypeScript down plugins.

## Plugins

- `ts-to-schema` - TypeScript to schema converter plugin

## Installation

```bash
bun install
```

## Development

```bash
bun run dev
```

## Build

```bash
bun run build
```

## Test

```bash
bun run test
```

## Lint

```bash
bun run lint
```
