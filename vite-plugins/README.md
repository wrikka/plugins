# Vite Plugins

Collection of Vite plugins for development.

## Plugins

- `wanalyzer` - Code analyzer plugin
- `wformatter` - Code formatter plugin
- `wlinter` - Code linter plugin
- `wmarkdownrender` - Markdown renderer plugin
- `wmock` - Mock data plugin
- `wnot-use` - Unused code detector plugin
- `wprecompile` - Precompile plugin
- `wpreview` - Preview plugin
- `wreplace` - Replace plugin
- `wrikka` - Rikka plugin
- `wscan` - Code scanner plugin
- `wstyles` - Styles plugin
- `wvitest-auto-import` - Vitest auto-import plugin
- `wvitest-benchmark` - Vitest benchmark plugin
- `wvitest-reporter` - Vitest reporter plugin

## Installation

```bash
bun install
```

## Development

```bash
bun run dev
```

## Build

```bash
bun run build
```

## Test

```bash
bun run test
```

## Lint

```bash
bun run lint
```
